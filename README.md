ECHO 42

Synopsis

En 1983, une guerre nucléaire éclate, tuant des milliers de civils. Seuls certains militaires proches des abris parviennent à survivre. Vingt ans plus tard, Ivanov Belinski, un ancien opérateur radio de l’URSS, sort d’un bunker militaire dans l’espoir de retrouver sa fille, dont il a été séparé. Au fil de ses recherches, il découvre un bunker émettant un signal de détresse. Sans autre piste, il s’y engouffre, espérant être enfin réuni avec elle.

Système d'interface évolutive

La particularité de Echo 42 réside dans son interface utilisateur dynamique. L'UI change radicalement pour s'adapter aux besoins d'Ivanov selon la situation.

Phase d'exploration

L'affichage est optimisé pour la reconnaissance et la gestion des ressources. Le joueur peut utiliser la radio pour :

  Scanner les pièces : Détecter les menaces en amont pour engager le combat avec un avantage tactique.
  Localiser des batteries : Identifier des sources d'énergie essentielles pour maintenir la radio opérationnelle.

Phase de combat

L'interface mute totalement pour se concentrer sur l'affrontement. Le combat repose sur l'observation et l'analyse :

  Analyse de pattern : Le joueur doit utiliser les cadrans de l'interface et examiner l'enemi pour battre celui-ci.
