<h1 align="center">ECHO 42</h1>

---

## SYNOPSIS
> En 1983, une guerre nucléaire éclate, tuant des milliers de civils. Seuls certains militaires proches des abris parviennent à survivre. 
> 
> Vingt ans plus tard, **Ivanov Belinski**, un ancien opérateur radio de l’URSS, sort d’un bunker militaire dans l’espoir de retrouver sa fille, dont il a été séparé. Au fil de ses recherches, il découvre un bunker émettant un signal de détresse. Sans autre piste, il s’y engouffre, espérant être enfin réuni avec elle.

---

## SYSTEME D'INTERFACE EVOLUTIVE
La particularité de **Echo 42** réside dans son interface utilisateur dynamique. L'UI change radicalement pour s'adapter aux besoins d'Ivanov selon la situation.

### PHASE D'EXPLORATION
L'affichage est optimisé pour la reconnaissance et la gestion des ressources. Le joueur utilise sa radio pour :
* **Scanner les pièces** : Détecter les menaces en amont pour engager le combat avec un avantage tactique.
* **Localiser des batteries** : Identifier des sources d'énergie essentielles pour maintenir la radio opérationnelle.

### PHASE DE COMBAT
L'interface mute totalement pour se concentrer sur l'affrontement. Le combat repose sur l'observation et l'analyse :
* **Analyse de pattern** : Le joueur doit manipuler les cadrans de l'interface et examiner précisément l'ennemi pour neutraliser la menace.
